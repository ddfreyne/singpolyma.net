--- title: Table for Converting Blogger Classic to New Blogger date: 2007-01-17 20:19:00 Z tags: ["Tech","Blogger","Template","Templates","Code"] wordpress_path: /2007/01/table-for-converting-blogger-classic-to --- I have added a lot of new tags and some corrections to the table. A few people have asked for something like this. A table of Blogger Classic template tags and their new equivalent. This is not entirely possible because it is not a 1:1 relationship, but I will do my best. This is a post for those who were somewhat literate in the old Blogger and want to upgrade. Another post to check out would be Johan's original post. Great stuff. If you have a template tag from Blogger Classic that you don't see here, post it in a comment and I'll try to add it. This is a work in progress.
