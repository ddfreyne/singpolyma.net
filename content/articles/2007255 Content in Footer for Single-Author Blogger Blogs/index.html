--- title: Content in Footer for Single-Author Blogger Blogs date: 2007-09-12 15:53:11 Z tags: ["Tech","Blogger","Hack","Hacks"] wordpress_path: /actionstream/content-in-footer-for-single-author-blogger-blogs --- Awhile ago I received this message from an unidentified user:
Title: photo for authors Can you use javascript to show an author's photo in the footer of a post? I've looked everywhere and only find it in the old blogger Any help would be awesome!
While a script based on my new comment author photos could be built to inject the image for the post author (multi-author blogs), I thought it might first be useful to provide basic information about hard-coding this.
  1. Go to the Template tab on your blog's dashboard
  2. Select Edit HTML
  3. Check the Expand Widget Templates box
  4. Find the code block starting with <div class='post-footer'> or after <data:post.body/>
  5. Insert the content you want there
  6. Save!
